Tradução **pt-BR** para **Starstorm 2** by **TeamMoonstorm**.

> 🇧🇷 Este pacote contém somente a tradução **Português (Brasil)**.

> Primeira tradução: Kauzok e Balorfist. Conversão da versão EN + tradução parcial anterior que não constava no mod. Revisado linha a linha por Jaosnake.

| Flag | Language |
|------|----------|
| 🇧🇷 | Portuguese (Brazil) / Português (Brasil) |

## 📥 Instalação Manual
1. Instale **Starstorm 2** e **R2API_Language**.
2. Baixe este pacote.
3. Copie a pasta `Jaosnake-Starstorm2_ptBR_LanguagePack` para `BepInEx/plugins/`.
4. No jogo, selecione **Português (Brasil)**.

Este pacote contém apenas arquivos de tradução. Ele não modifica o código do Starstorm 2.
