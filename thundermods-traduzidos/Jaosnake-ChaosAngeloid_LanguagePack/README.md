# ChaosAngeloid Language Pack

This pack provides multi-language support for the ChaosAngeloid mod.

## Usage

1. Install the ChaosAngeloid mod
2. Install this language pack
3. Select your preferred language in the Risk of Options menu

## Languages

- English (en) - Original
- Esperanto (eo) - Translated
- Latin (la) - Translated

## Translation Notes

Esperanto uses x-system notation (cx, sx, gx, jx, ux, hx).
Latin uses classical Latin with modern game terminology.
