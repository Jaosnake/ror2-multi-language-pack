# Sandswept Language Pack

## Overview
Multi-language translation pack for **Sandswept** (by SandsweptTeam).  
Supports **13 languages** with full coverage of all tokens (items, equipment, survivors, skills, enemies, shrines, drones, interactables, elite variants).

## Installation (Thunderstore)
1. Download this package from Thunderstore.
2. Install with r2modman or place in `BepInEx/plugins/`.
3. Ensure **Sandswept** (SandsweptTeam-Sandswept) is installed.
