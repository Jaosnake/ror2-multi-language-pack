Tradução **pt-BR** para **Sandswept**, feito por **SandsweptTeam**.

> 🇧🇷 Este pacote contém somente a tradução **Português (Brasil)**.

> Conversão direta da versão EN, pois não havia suporte pt-BR. Conteúdo traduzido 100% à mão por Jaosnake.

## 📥 Instalação Manual
1. Instale **Sandswept** e **R2API_Language**.
2. Baixe este pacote.
3. Copie a pasta `Jaosnake-Sandswept_ptBR_LanguagePack` para `BepInEx/plugins/`.
4. No jogo, selecione **Português (Brasil)**.

Este pacote contém apenas arquivos de tradução. Ele não modifica o código do Sandswept.
